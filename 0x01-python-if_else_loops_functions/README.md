The following is to illustrate the basic usage of if/else statements, loops, and functions in Python with code examples.

## If/Else Statements

Use `if`, `elif`, and `else` for conditional statements:

```python
x = 10

if x > 10:
    print("x is greater than 10")
elif x == 10:
    print("x is equal to 10")
else:
    print("x is less than 10")
```

## For Loop
Iterate through sequences or collections:
```python
fruits = ["apple", "banana", "orange"]

for fruit in fruits:
    print(fruit)
```

## While Loop
Execute code while a condition is true:
```python
count = 0

while count < 5:
    print(count)
    count += 1
```

## Functions
Define reusable blocks of code:

```python
def greet(name):
    return f"Hello, {name}!"

result = greet("Alice")
print(result)
```
